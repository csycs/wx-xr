# AR 扫描识别小程序 - 设计风格指南

> 基于参考电商界面的设计系统

## 🎨 设计系统概述

本设计系统参考了现代电商应用的设计风格，采用蓝色主题色，清爽的卡片式布局和清晰的信息层级。

---

## 🎨 颜色系统

### 主色调

| 颜色名称 | 色值 | 使用场景 |
|---------|------|---------|
| 主色 | `#007AFF` | 导航栏选中、按钮、链接、激活状态 |
| 主色浅 | `#40A0FF` | 悬停状态 |
| 主色深 | `#0056CC` | 按下状态 |

## 🧩 组件库

### StatusBadge 状态徽章

用于显示不同状态的徽章组件。

```vue
<StatusBadge type="generating" text="生成中" />
<StatusBadge type="queueing" text="排队中" />
<StatusBadge type="success" text="完成" />
<StatusBadge type="warning" text="警告" />
```

### ProductCard 产品卡片

用于展示产品信息的卡片组件。

```vue
<ProductCard
  image="/path/to/image.jpg"
  title="产品标题"
  status="生成中"
  statusType="generating"
  time="2分钟前"
  description="产品描述"
  price="¥99"
  @click="handleClick"
/>
```

## 📱 页面布局

### 顶部标题区

- 背景：白色
- 内边距：`24px 16px 16px`
- 标题：`18px Bold`
- 副标题：`12px` 次要文本色

### 标签导航

- 背景：白色
- 粘性定位
- 激活状态：蓝色文字 + 下划线
- 未激活：灰色文字

### 内容区

- 内边距：`12px`
- 卡片间距：`12px`

---

## 🎯 交互状态

### 按钮

- 默认：主色背景
- 悬停：透明度 `0.8`
- 按下：轻微缩放 `0.98`

### 卡片

- 默认：轻微阴影
- 点击：阴影加深 + 缩放

### 输入框

- 默认：灰色边框
- 聚焦：蓝色边框 + 白色背景

## 📋 使用指南

### 导入设计变量

```scss
@import '@/styles/design-tokens.scss';

.my-element {
  color: $primary-color;
  padding: $spacing-lg;
  border-radius: $radius-md;
}
```

### 使用全局类

```vue
<view class="page-container">
  <text class="text-primary font-bold">主文本</text>
  <text class="text-secondary">次要文本</text>
</view>
```

## 📝 注意事项

1. **统一使用设计变量**：所有颜色、间距、字体都应使用设计系统变量
2. **保持视觉一致性**：相同的元素应使用相同的样式
3. **响应式设计**：注意不同屏幕尺寸的适配
4. **性能优化**：合理使用 `v-if` 和 `v-show`
5. **无障碍**：确保足够的颜色对比度
